# Budding Yeast dataset

It contains the expression levels of 2467 genes on 79 samples
corresponding to 8 different experiments of the budding yeast: alpha
factor (18 samples), cdc15 (15 samples), cold shock (4 samples), diauxic
shift (7 samples), DTT shock (4 samples), elutriation (14 samples), heat
shock (6 samples) and sporulation (11 samples).

## Usage

``` r
data(dsyeastexpression)
```

## Format

An data frame with 2467 rows and 79 columns.

## Source

<https://www.pnas.org/content/suppl/1998/12/08/95.25.14863.DC1/3917data.xls>

## Details

Once the data set is loaded, it can be accessed as an object of class
dataframe called `dsyeastexpression`.
